# Video Call App

Simple WebRTC video calling app using Node.js, Express, and Socket.IO for signaling.

## Setup
```bash
npm install
npm start
```
Open http://localhost:3000 in two tabs/devices, enter the same Room ID, and join.

## Files
- `server.js` — Express + Socket.IO signaling server
- `public/index.html` — Frontend UI
- `public/script.js` — WebRTC + signaling logic
- `public/style.css` — Styles
